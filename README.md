## ✨ Features

- 👤 **Professional Identity** — AI-enhanced diagnostic portrait and detailed clinical background of SLP Sanjay Kumar.
- ✨ **Premium UI/UX** — Custom glowing cursor, 3D parallax tilt effects, and smooth Framer Motion scroll animations.
- 📹 **Video Case Studies** — High-impact "Before & After" video modules demonstrating real patient success stories.
- 🛠️ **Clinical Services** — Detailed showcase of Voice Therapy, Stammering, Hearing Aids, and Pediatric Speech services.
- 📬 **Interactive Reach Us** — Integrated high-definition Google Maps and authoritative clinic contact information.
- 📱 **Multi-Device Support** — Fully responsive architecture that maintains a premium look on mobile, tablet, and desktop.
- ⚡ **Edge Performance** — Hosted on Firebase CDN for instantaneous loading across the globe.

## 🚀 Getting Started

To run the clinic website locally:

```bash
# 1. Clone the repository
git clone https://github.com/Narasimha-jampana/SNL-Webpage.git

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

---

## 🌍 Deployment

This project is deployed using **Firebase Hosting**.

```bash
# Build the production bundle
npm run build

# Deploy to the live site
firebase deploy --only hosting:snl-voice
```
